## ❓ Still Unsure Which License You Need?

Ask yourself these questions in order:

1. **Do you earn any money using this software?**
   - YES → Commercial License
   - NO → Next question

2. **Are you a registered business (any type)?**
   - YES → Commercial License
   - NO → Next question

3. **Is this for learning, a pet project, or non-profit?**
   - YES → Community License (FREE)
   - NO → Commercial License
